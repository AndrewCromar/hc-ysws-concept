# Andrew's Hack Club YSWS Idea

Build your own video-game controller.

## Core Idea

Students spend time designing a PCB for their game controller using parts provided by the program apon completion.
Students design a 3d printed housing to enclose the controller and pcb.
Students write code for a microcontroller to emulate a hid controller.

When all the parts are done students can submit their designes to the program to have their parts manufatured/printed and shipped to them.

Students will assemble and test their controller. Then they will create a ship post for the Hack Club slack and request to be added to the gallary.

Boom students learn how to make video game controllers.

## More Info

Detailed instructions will be provided for students to start from the ground up to have a finished product.

Additional challenges for more prizes can exist (ex. make it wireless).

Students will make organized and well documented github repos for submission (structure ex. assets, firmware, pcb, cad, production, readme).
